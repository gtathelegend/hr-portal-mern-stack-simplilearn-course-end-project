# HR Portal

A React-based HR portal application for managing employees and leaves. This is a frontend-only application that simulates a full MERN stack application, with all data managed in-memory.

## Features

-   **User Authentication:** Login and Signup functionality for two distinct user roles:
    -   HR Manager
    -   Employee
-   **HR Dashboard:**
    -   Add new employees to the system.
    -   View a comprehensive list of all employees.
    -   Review, approve, or reject leave requests submitted by employees.
-   **Employee Dashboard:**
    -   Submit leave applications.
    -   View personal leave history and the status of each application (Pending, Approved, Rejected).

## Technologies Used

-   **Frontend:**
    -   [React](https://reactjs.org/)
    -   [Bootstrap](https://getbootstrap.com/)
-   **Programming Language:**
    -   JavaScript (ES6+)
-   **Styling:**
    -   HTML5 & CSS3

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your system.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/hr-portal-mern-stack-simplilearn-course-end-project.git
    cd hr-portal-mern-stack-simplilearn-course-end-project
    ```
2.  **Install NPM packages:**
    ```sh
    npm install
    ```

### Running the Application

To start the development server, run the following command:

```sh
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
